# 任务管理系统

一个基于Web的任务管理系统，用于帮助学生管理学习任务和课程进度。

## 功能特性

- 📋 **任务管理**：创建、编辑、删除和标记任务完成状态
- 📚 **课程管理**：添加、编辑和删除课程，为每个课程设置不同的颜色标识
- 📊 **进度查看**：实时查看学习进度和任务完成情况
- 📈 **统计报告**：生成详细的学习统计报告
- 📱 **响应式设计**：支持桌面和移动设备
- 💾 **本地存储**：数据保存在浏览器本地存储中

## 技术栈

- **前端框架**：HTML + CSS + JavaScript
- **UI组件库**：Tailwind CSS v3
- **图标库**：Font Awesome
- **数据存储**：LocalStorage

## 快速开始

### 方法一：直接打开HTML文件（推荐）
1. 双击项目文件夹中的 `index_premium.html` 文件
2. 或者在浏览器中打开 `index_premium.html` 文件

### 方法二：使用启动脚本
1. 双击运行 `启动应用.bat` 文件
2. 系统将自动在默认浏览器中打开应用程序

### 方法三：使用本地服务器（高级用户）
#### 选项A：使用Python
1. 安装Python（如果尚未安装）
2. 在项目目录中运行命令：`python -m http.server 8000`
3. 在浏览器中访问：`http://localhost:8000`

#### 选项B：使用Node.js
1. 安装Node.js（如果尚未安装）
2. 双击运行 `启动Node服务器.bat` 文件
3. 或者在项目目录中运行命令：`node server.js`
4. 在浏览器中访问：`http://localhost:8000`

## 版本说明

本项目提供多个界面版本，满足不同用户的需求：

- **index_solid_bg.html** - 纯色背景版界面（默认）：采用简洁的纯色背景设计，提供更清爽的视觉体验，适合喜欢简约风格的用户。包含四种背景选项（纯色、渐变、图片、图案），可通过右上角的圆形按钮切换。

- **index_premium.html** - 高级版界面：采用现代化设计，包含渐变背景、玻璃态效果、动画过渡和精美的卡片布局，提供丰富的视觉体验。

- **index_offline.html** - 离线版界面：完全离线运行，不依赖任何外部资源，适合网络环境受限的情况。

- **index.html** - 原始版界面：使用CDN资源的原始版本。

## 使用指南

### 添加课程
1. 点击顶部导航栏中的「课程管理」
2. 点击「添加课程」按钮
3. 输入课程名称并选择颜色
4. 点击「保存」

### 创建任务
1. 点击顶部导航栏中的「任务列表」
2. 点击「添加任务」按钮
3. 输入任务标题、选择关联课程、设置截止日期
4. 点击「保存」

### 查看进度
1. 点击顶部导航栏中的「进度查看」
2. 查看总体进度和各课程进度
3. 查看近期截止的任务

### 生成报告
1. 点击顶部导航栏中的「生成报告」
2. 查看详细的统计数据
3. 点击「导出报告」可导出数据（模拟功能）
